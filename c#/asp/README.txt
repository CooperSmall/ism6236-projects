Installation Instructions
_________________________

1) Open 'smallasg5.sln' from the smallasg5 folder using Visual Studio
2) Check that 'smalldblib' is listed under the References folder
3) If not, add a reference to the 'smalldblib' project from the smalldblib folder
4) Build and Deploy 'smallasg5' using IIS Express
5) Click 'List Reservations' to access smalldblib's listreservation method
6) Click 'Book Reservation' to access smalldblib's listavailable and book methods
