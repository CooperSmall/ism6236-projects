Installation Instructions
_________________________

1) Open 'smallcooperasg4.sln' using Visual Studio
2) Open Solutions Explorer
3) R-Click 'smalldblib' and select build
4) Collapse 'Client' and R-Click 'References'
5) Select 'Add Reference'
6) Check'smalldblib' from the list and select 'Ok'
7) R-Click 'Client' and select build
8) Select 'Start' on the top ribbon
9) Enjoy
